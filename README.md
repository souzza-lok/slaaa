O Bitbucket é uma plataforma de hospedagem de código-fonte baseada em Git, focada em equipes de desenvolvimento para armazenar, gerenciar e revisar projetos. Criado pela Atlassian, ele se destaca pela integração com Jira e Trello, sendo ideal para repositórios privados e colaboração profissional. Oferece CI/CD integrado (Pipelines) e controle de versão


Um Issue Tracker (rastreador de problemas) é uma ferramenta de software utilizada para registrar, organizar, priorizar e monitorar o status de bugs, tarefas e solicitações de melhorias em projetos, sendo essencial para o desenvolvimento de software e suporte ao cliente. Ele permite que equipes acompanhem o progresso de tarefas, atribuam responsáveis e documentem o fluxo de trabalho até a resolução final




Uma pull request (solicitação de pull ou "PR") é uma proposta no GitHub e outros sistemas Git para mesclar alterações de um branch para outro. Ela permite que a equipe revise, comente e discuta o código antes da sua integração na base principal, garantindo qualidade e colaboração. A PR mostra as diferenças (diffs) entre o código original e o proposto
